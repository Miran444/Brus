## Strojne zahteve

- ESP32-S3 DevKit
- USB kabel

## Programske zahteve

- PlatformIO
- Visual Studio Code

## Navodila za uporabo

1. Odprite projekt v VS Code s PlatformIO
2. Priključite ESP32-S3 na računalnik
3. Pritisnite `PlatformIO: Upload` za nalaganje kode
4. Odprite `PlatformIO: Serial Monitor` za ogled izpisa
